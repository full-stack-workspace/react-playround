### Git Workflow

AI programming tools MUST follow the git workflow defined in `openspec/specs/git-workflow/spec.md`.

**Key Rules:**
- Generate semantic commit messages using Conventional Commits format
- Always get user confirmation before committing
- Use `git pull --rebase` before pushing
- Run tests and build checks before push
- Handle conflicts by notifying the user, not auto-resolving
